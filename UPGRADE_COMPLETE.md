## 🚀 Getting Started

### 1. Quick Start (60 seconds)
```bash
git checkout pro-upgrade
python -m venv venv
.\venv\Scripts\activate  # or: source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your API keys
python main.py
```

### 2. Configure API Keys

Edit `.env`:
```ini
# Get free keys:
OPENAI_API_KEY=sk-...        # openai.com (free trial)
GEMINI_API_KEY=...           # makersuite.google.com (free)
SEARCH_API_KEY=...           # serpapi.com (100 free/month)
```

### 3. Run JARVIS Pro

```bash
# GUI Mode (recommended)
python main.py

# CLI Mode
python main.py --cli
```

### 4. First Test

Type or speak:
```
"Hello Jarvis"
"What time is it?"
"Tell me about Python 3.12"
"What's the weather?"
"Open notepad"
```
